# Meta: как устроен проект

В этом разделе собираются документы о самой базе знаний SafeVibe:

- Архитектура файлов и разделов.
- Таксономия типов заметок и тегов.
- Правила для контента и мини-апп.
- Рекомендации по контрибьюции.

Подробнее смотри:

- `meta/architecture` — архитектура и структура.
- `meta/taxonomy` — категории, типы и теги.
- `meta/content-style` — стиль статей.
- `meta/contributing` — как вносить вклад.
