# Воркфлоу вайб-кодера

В этом разделе собираются рабочие процессы:

- SDD (Scenario-Driven Development / сценарный дизайн).
- TDD (Test-Driven Development).
- Design Thinking.

Каждый из подпроцессов будет представлен как отдельный плейбук с шагами и примерами.
