# Инструменты вайб-кодера

Этот раздел — хаб по инструментам, с которыми работает вайб-кодер.

Подразделы:

- **IDE** — редакторы и IDE (Cursor, Windsurf, Zed, и др.).
- **CLI** — инструменты командной строки (Claude Code, OpenCode, Qwen CLI, Gemini CLI, и др.).
- **LLM** — модели и их клиенты (GPT, Claude, GLM, Qwen, Gemini, и др.).
- **MCP** — провайдеры контекста и расширений (Context7, Claude Context, Sequential, и др.).
- **Сервисы** — внешние сервисы (AI Studio, Z AI, Perplexity, провайдеры и др.).

В следующих итерациях здесь появятся карточки-плитки с фильтрами по типу инструмента, LLM, платформе и теги из front matter.
