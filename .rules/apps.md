# Правила для мини-апп

> Черновик. Требования к мини-аппам (React + shadcn + MDX).

- Все мини-аппы живут в `src/apps/<app-name>/`.
- UI строится на компонентах из `src/components/ui/*` (shadcn).
- Каждая мини-аппа встраивается через MDX-страницу в `docs/apps/`.
- Обязательные состояния: loading, empty, error.

Детализация будет расширяться по мере появления первых мини-апп.
